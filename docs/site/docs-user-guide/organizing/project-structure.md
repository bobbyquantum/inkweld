# The Project Interface

When you open a project, you'll see Inkweld's project interface. Let's explore each area and how they work together.

## Overview

<ThemedImage
  src="/img/generated/project-home-desktop"
  alt="Inkweld project interface showing the project home page"
/>

The interface is divided into three main areas:

1. **Tab Bar** (top) ‚Äî Navigate between views including the home tab and open documents
2. **Project Navigator** (left) ‚Äî Browse your project's structure
3. **Content Area** (center) ‚Äî View and edit content

## The Tab Bar

The tab bar runs across the top of the interface and provides navigation between different views.

### Home Tab

The first tab (with a home icon) takes you to the **Project Home** page, which displays:

- **Project header** ‚Äî Cover image, title, and description (click to edit)
- **Start actions** ‚Äî Quick buttons for common tasks like creating files or exporting
- **Recent elements** ‚Äî Documents you've opened recently for quick access
- **Publish plans** ‚Äî Manage your EPUB/PDF export configurations

### Document Tabs

When you open documents, they appear as additional tabs. You can:

- **Click a tab** to switch between open documents
- **Click the X** on a tab to close it
- **Middle-click** a tab to close it quickly
- **Right-click** a tab for more options

#### Tab Context Menu

Right-click any tab to access the context menu:

<ThemedImage
  src="/img/generated/tab-context-menu"
  alt="Tab context menu showing close options"
/>

- **Rename** ‚Äî Rename the element directly from the tab (only appears for documents, folders, and worldbuilding elements)
- **Close** ‚Äî Close this tab
- **Close to Right** ‚Äî Close all tabs to the right of this one
- **Close Others** ‚Äî Close all other tabs except this one
- **Close All** ‚Äî Close all open tabs

### Toolbar Actions

On the right side of the tab bar, you'll find:

- **More menu** (‚ãÆ) ‚Äî Additional actions like export, import, and project deletion
- **User menu** ‚Äî Access your account, settings, and logout

### Navigator Actions

- **Click an element** to open it in a new tab
- **Drag and drop** to reorganize elements
- **Right-click** for context menu (rename, delete)
- **Click the chevron** to expand/collapse folders

### Navigator Toolbar

At the top of the navigator, you'll find:

- **Home button** ‚Äî Return to the project home tab
- **New Document** ‚Äî Create a new document at the root level
- **New Folder** ‚Äî Create a new folder at the root level

### Bottom Navigation

At the bottom of the navigator, you'll find quick access to:

- **Media Library** ‚Äî Manage project images and files
- **Project Settings** ‚Äî Configure project options
- **Exit Project** ‚Äî Return to the bookshelf

## User Menu

The user menu appears in the top-right corner of the interface. Click your avatar (or initials) to access:

- **Messages** ‚Äî View system announcements
- **Settings** ‚Äî Configure application preferences (connection, display options)
- **Admin** ‚Äî Access admin dashboard (if you're an administrator)
- **Logout** ‚Äî Sign out of your account

## Mobile Layout

On mobile devices, the interface adapts for touch:

- **Hamburger menu** opens the project navigator as a slide-out panel
- The navigator includes all navigation options in one place
- Swipe or tap the backdrop to close the panel
- The content area takes the full screen width
